# 工程管理システム v1.00

本ツールは、エンジニアリング業務の工程管理・リソース配分を直感的に行うためのWebアプリケーションです。インストール不要で、ブラウザだけで動作します。

## 🌟 主な特徴
- **高速工程編集**: マウス操作でバーをクリック・ドラッグするだけで工程の割り当てや期間調整が可能です。
- **製番・案件リスト管理**: Excelからリストをコピー＆ペーストするだけで、大量の案件データを一括で取り込めます（Ctrl + V）。
- **FHD最適化レイアウト**: 1920x1080の画面に最適化され、スクロールなしで多くの情報を俯瞰できます。
- **データのポータビリティ**: JSON形式でのエクスポート・インポートに対応。バックアップやチーム内での共有が容易です。
- **印刷専用レイアウト**: A4縦向き、白黒のシンプルな帳票形式での出力に対応しています（1ページ目に工程表、2ページ目以降に製番リスト）。

## 🚀 使い方
1. **起動**: `index.html` をブラウザ（Google Chrome, Microsoft Edge推奨）で開きます。
2. **案件の取り込み**:
   - 「製番リスト」ボタンを押し、Excelの表（指令書No.、客先、製品名など）をコピーして、画面上の「ここに貼り付け」エリアで `Ctrl + V` を押します。
3. **工程の割り当て**:
   - 工程表のセルをクリックして、「製番の割当」または「休暇・特別項目」を選んで保存します。
   - 割り当てられたバーは、左右の端をドラッグして期間変更、中央をドラッグして移動が可能です。
4. **データの保存**:
   - 右上の「データ出力」からJSONファイルを保存できます。
   - 保存したファイルを画面にドラッグ＆ドロップすると、データを復元できます。
5. **印刷**:
   - ヘッダーの「印刷」ボタンを押すと、印刷プレビューが開きます。

## ⚠️ 注意事項
- **データ保存**: 入力したデータはブラウザの「ローカルストレージ（記録領域）」に保存されます。ブラウザのキャッシュを完全に消去するとデータも消えるため、**定期的に「データ出力」でバックアップを保存すること**を強く推奨します。
- **ブラウザ**: 最新の Google Chrome または Microsoft Edge での使用を前提としています。

## 🛠 ファイル構成
- `index.html`: アプリケーションの本体
- `app.js`: システムの制御ロジック
- `style.css`: デザイン・印刷レイアウト定義
