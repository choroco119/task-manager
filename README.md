# 工程管理システム v1.00

本ツールは、エンジニアリング業務の工程管理・リソース配分を直感的に行うためのWebアプリケーションです。
ブラウザと接続した共有フォルダ間で直接データを読み書きでき、週別での工程・スケジュール管理が可能です。

---

## 🌟 主な特徴

- **週別データ管理**: 週の頭（月曜日）の日付を基準に、`data_yyyymmdd.json` 形式で週ごとにデータを自動で保存・切り替えします。
- **高速工程編集**: マウス操作でバーをクリック・ドラッグするだけで工程の割り当てや期間調整が可能です。
- **製番・案件リスト管理**: Excelからリストをコピー＆ペーストするだけで、大量の案件データを一括で取り込めます（Ctrl + V）。
- **FHD最適化レイアウト**: 1920x1080の画面に最適化され、スクロールなしで多くの情報を俯瞰できます。
- **データの安全な同期**: File System Access API による共有フォルダ直接書き込みと排他制御（ロック機能）により、複数人での競合を防ぎます。
- **自動バックアップ**: 共有フォルダ内の `backups` ディレクトリに、過去30日分のデイリーバックアップを自動で保存します。
- **印刷専用レイアウト**: A4縦向きでの出力に対応しています（1ページ目に工程表、2ページ目以降に製番リスト）。

---

## 🚀 使い方

1. **接続先フォルダの設定**:
   - 右上の「同期設定」ボタンを押し、データを保存するフォルダ（共有フォルダなど）を選択・アクセス権限を許可します。
   - 次回以降は「クイック同期」ボタンで以前選択したフォルダに即座に接続できます。
2. **週の移動・作成**:
   - 「前週」「次週」ボタンで週を変更します。
   - 変更先にデータが存在しない場合は、「現在の週のデータを引き継いで作成するか」確認ダイアログが表示されます。OKを押すと引き継いで新規作成され、キャンセルを押すと切り替えを中止します。
3. **案件の取り込み**:
   - 「製番リスト」ボタンを押し、Excelの表をコピーして、画面上の「ここに貼り付け」エリアで `Ctrl + V` を押します。
4. **工程の割り当て**:
   - 工程表のセルをクリックして、「製番の割当」または「休暇・特別項目」を選んで保存します。
   - 割り当てられたバーは、左右の端をドラッグして期間変更、中央をドラッグして移動が可能です。
5. **データの更新と自動同期**:
   - 編集内容は自動で週別ファイルに保存されます。
   - 他のユーザーがデータを更新した場合は、自動的に検知して画面に反映されます（右上の「更新」ボタンで手動リロードも可能です）。

## 🛠 ファイル構成

- `index.html`: アプリケーションのメイン画面・構造定義
- `app.js`: アプリケーションのメイン制御ロジック、描画制御
- `js/sync-manager.js`: フォルダ接続・週別ファイル読み書き・自動同期・ロック・バックアップなどの同期ロジック
- `style.css`: デザイン、カラーパレット、レスポンシブ・印刷用レイアウト定義

---

## ⚠️ 注意事項

- **ブラウザ制限**: File System Access API を使用するため、最新の Google Chrome または Microsoft Edge が必要です。また、セキュリティの都合上、HTTPS経由または localhost でのみフォルダ接続機能が有効になります。
- **データの整合性**: 週別ファイルには、その週のスケジュール、プロジェクト定義、スタッフ定義など、その週で表示されるすべてのデータが保存されます。
