# ワークスルー：工数集計機能の実装

リソース管理の効率化のため、個人およびチーム単位での工数集計機能を追加しました。

## 新機能の概要

### 1. 個人別「合計工数」列
- 工程表の右端に「合計工数」列を新設。
- その週に担当者が割り当てられている全工程の工数を自動合算。
- 工数データ形式（"30H"、"30Hx2"等）に応じた高度なパース処理を搭載。

### 2. チーム別統計エリア (v2.90)
- 工程表の下部に、チーム全体の稼働状況を示すサマリーカードを表示。
- **合計工数**: チームメンバー全員の合計時間。
- **平均工数**: 1人あたりの平均稼働時間（合計 / メンバー数）。
- チームフィルターと連動し、特定のチームのみの統計を確認可能。

## 検証結果
- 各担当者の右端に正しい合計工数が表示されることを確認。
- 下部の統計エリアが、読み込まれたデータおよびフィルター状態に応じて正確に更新されることを確認。
- 工数が未入力のプロジェクトがある場合でも、合算時にエラーにならないことを確認。
