# ワークスルー：工程割り当てフローの改善

工程割り当ての操作性を向上させるため、同一ウィンドウ内でのステップバイステップ方式を導入しました。

## 変更のハイライト

### 1. ステップバイステップの割り当て
空のセルをクリックすると、まず製番リストが開きます。製番を選択すると、自動的に曜日（期間）の調整画面へ遷移します。

### 2. 同一ウィンドウ内での完結
以前は別ウィンドウが開いていましたが、これをモーダル表示に変更。視線の移動を減らし、直感的な操作が可能になりました。

### 3. モーダルの最適化
製番リストは情報量が多いため、画面の 95% を使用する大型モーダルとして実装しました。
