# 工程管理システム v1.00 正式版 リリースノート

本バージョンは、現場での実用性と、大画面（FHD）環境での圧倒的な情報集約力を追求した「完成版」です。

## 主要機能一覧

### 1. ユーザーインターフェース (UI/UX)
- **FHDスクロールレス・デザイン**: 1920pxの横幅をパズルのように最適配分し、主要情報を一画面に収容。
- **高精度テーブルレイアウト**: チーム・担当者列に絶対値を割り当て、ブラウザの自動縮小に負けない視認性を確保。
- **プレミアム・ダークテーマ**: 長時間の作業でも疲れにくい、モダンな高コントラスト配色。

### 2. 工程割り当てロジック
- **ステップバイステップ・モーダル**: 迷わせないガイダンス方式の割当フロー。
- **リバース・アサイン・フロー**: 「どの製番を、どこに？」という逆引きの操作に対応。
- **高速データ入力**: 1文字ごとの再描画を排除し、Excelのような軽快なキー入力を実現。

### 3. データ分析・可視化
- **週計工数カウンター**: 各メンバーの右端に、その週の合計工数をリアルタイム表示。
- **チーム別ダッシュボード**: 工程表の下部に、チームごとのリソース合計と平均稼働状況を可視化。

### 4. システム堅牢性
- **別ウィンドウモード対応**: 製番リストをサブモニターで常時表示可能な独立駆動モード。
- **Excel貼り付け同期**: 貼り付けたデータの自動解析と、IDの自動紐付け機能。

## バージョン履歴
- `v1.00`: 初回正式版公開。すべてのUI崩れおよび入力不整合の解消。
- `v0.9x`: FHD最適化、工数集計、逆引きフローの試行実装。
