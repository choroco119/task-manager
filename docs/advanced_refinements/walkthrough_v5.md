# 完了報告: リスト連動と手入力追加機能のご要望対応

ご提示いただいた下記2点のご要望について対応が完了いたしました。
ブラウザを更新（リロード）して動作をご確認ください。

## 1. バー表示テキストを「リストNo.」に変更
*   **変更内容**: 工程バーの中に表示される先頭の番号を、「[指令書No]」から「リスト上の並び順の [No.]」へと変更しました。
*   **効果**: 今見ている予定バーが「製番・案件リスト」のどの位置（何行目）にあるのかを、視覚的に探しやすくなります。もちろんリスト上で並び順(No)を変更した場合は、連動してバーの表示番号も自動更新されます。

## 2. 製番・案件の手入力機能を追加
*   **変更内容**: 画面上部の歯車アイコンから開ける「製番・案件リスト管理」画面の右上に、青色の**「+手入力で追加」ボタン**を新設しました。
*   **効果**: Excelからコピー＆ペーストしなくても、ボタンを1回クリックするだけでリストの一番下に「空の新規製番行」が追加されるようになります。その後、追加された行の各項目（客先、製品名など）を直接クリックして手入力で編集していただけます。
