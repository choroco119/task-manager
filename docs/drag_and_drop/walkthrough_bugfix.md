# 修正報告: JSONファイルドロップ画面の誤作動修正

バーをドラッグした際に「JSONファイルをドロップして読み込み」という大きな外部ファイル用の画面が誤って全面を覆ってしまう不具合を修正しました。

## 原因と修正内容
*   **事象**: 本来はパソコンのフォルダから手動バックアップ用などの「JSONファイル」をブラウザに投げ込んだ時だけに出るべき画面が、内部のバーをドラッグしたときにも反応（誤爆）してしまっていました。
*   **対策**: ブラウザ側で検知するイベントに対して、「今マウスが掴んでいるものは外部からの**ファイル（Files）**か？ それとも画面内の要素か？」を確認するフィルター（`isFileDrag`）を追加しました。

これにより、内部のバーをドラッグしている間はJSONファイル読み込み用の画面が表示されず、快適に工程の移動を行っていただけるようになっています。
