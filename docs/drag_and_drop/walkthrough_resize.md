# 完了報告: バーの端部リサイズ機能の追加

スケジュールバーの左右の端をドラッグして、直感的に期間（日数）を伸縮できる機能を実装しました。

## 実装された機能
1. **ホバー時の視覚的サポート**
   *   すべてのバー（製番・特別項目）の「左端」および「右端」にマウスポインタを乗せると、カーソルが「左右両矢印（↔）」になリます。
   *   また、掴めるポイントに薄くハイライトがかかるため、直感的に操作可能な領域がわかります。
2. **リアルタイムな期間伸縮**
   *   端を掴んだままカーソルを左右に動かすと、現在のマウス位置（通り過ぎた日数のマス）を計算し、瞬時にバーの幅が追従して伸縮します。
3. **限界値の自動保護**
   *   右端を左に押し込みすぎて「開始日」を追い越したり、左端を右に引っ張りすぎて「終了日」を超えてしまうような「過去への逆転現象」が起きないよう、ストッパー（限界値制御）をプログラムに組み込んでいます。
   *   左に伸ばしすぎても月曜日(マス幅の左端)、右に伸ばしすぎても日曜日(マス幅の右端)でピタリと止まるよう補正されます。
4. **イベントのケンカ防止**
   *   「端をドラッグする操作」と「全体をドラッグして移動する操作」、そして「クリックしてダイアログを開く操作」が綺麗に独立して動作するように、内部でマウスイベントの遮断処理（`stopPropagation`, `preventDefault`）を入れています。

バーの端を掴んで長さを自由に変えられるか、ぜひブラウザをリロードして動作をご確認ください！
