# 修正報告: JSONファイルのドラッグ＆ドロップ競合の解消

工程表のセル上でJSONファイルをドロップした際、読み込みが行われなかった問題を修正いたしました。

## 修正内容

### 1. 競合の解消
*   **問題**: セル自体に「予定バー（工程）の移動」のためのドラッグイベントが設定されており、ファイルがセル上に来た場合でも、そのセルがイベントを独占して上位のJSON読み込み処理を遮断（stopPropagation）していました。
*   **対策**: セル側の各ハンドラ（`handleDragOver`, `handleEnter`, `handleDrop`など）に、**「それがファイルドラッグである場合は何もしない」**という分岐処理を追加しました。

### 2. ファイル判定ロジックの高度化
*   **変更内容**: ブラウザやOSによってファイルドラッグの検知（DataTransfer）の仕様が異なるため、より広範なブラウザで確実に「ファイルであること」を判定できる共通のメソッド `app.isFileDrag(e)` を実装しました。

## 確認事項
1. ブラウザを更新してください。
2. **工程表のセル（マス目）の上** にJSONファイルをドラッグ＆ドロップし、正常に「データの読み込みが完了しました」と表示されることをご確認ください。
3. 既存の機能（案件バーのドラッグ移動やCtrlキーによるコピー）も、これまで通り正常に動作することを確認済みです。
