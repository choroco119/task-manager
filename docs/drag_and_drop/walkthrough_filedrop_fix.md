# 修正報告: JSONファイルのドラッグ＆ドロップ読込修正

## 修正内容
*   **事象**: 外部からJSONファイルをドラッグ&ドロップしても、読み込み画面が出ない/読み込めないという事象を修正しました。
*   **原因**: 前回の「バーのドラッグとファイルのドラッグを区別するフィルター」において、ご使用のブラウザ環境によっては「マウスが持っているデータがファイルかどうか」の検知処理に対応できず、そのまま無視してしまっていたためです。
*   **対策**: あらゆるブラウザ環境に対応できるよう、ファイル属性の検知ロジックをより低レベルで強固なものに書き換えました。

お手数ですが再度ブラウザをリロードいただき、JSONファイルをD&Dして読み取り画面が正常に反応し、適用されるかお試しください！
