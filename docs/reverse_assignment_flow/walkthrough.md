# ワークスルー：逆順割り当てフローの実装

ユーザーの好みに応じて、「セルを先に選ぶ」方法と「製番を先に選ぶ」方法の両方のフローに対応しました。

## 新機能の概要

### 1. 製番先行選択モード (v2.95)
- 製番リスト（別窓またはモーダル）で「選択」を押した際、すでに特定のセルが選ばれていない場合、システムは「セル選択待ち」状態に入ります。
- 画面上部に **「製番 [XXX] を割り当てるセルを選択してください」** という案内バーが表示されます。

### 2. インタラクティブな位置指定
- 案内バーが表示されている状態で工程表のセルをクリックすると、選択済みの製番がその位置に即座に紐付けられます。
- その後、自動的に期間設定（曜日選択）のダイアログが開くため、一貫した操作感を提供します。

### 3. モードの解除
- 「キャンセル」ボタンを押すことで、いつでも割り当てを中止して通常モードに戻ることができます。
