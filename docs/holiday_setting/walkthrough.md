# 変更内容の説明: 工程表への休日設定機能

工程表（メイン画面）において、特定の日付を休日として設定し、視覚的に強調する機能を実装しました。

## 実施内容

### 1. 休日設定の操作 (日付ヘッダー)
- **トグル操作**: 工程表の上部にある日付（「4/11」など）をクリックすることで、その日を「休日」に設定できるようにしました。再度クリックすると解除されます。
- **視覚的なフィードバック**: 休日に設定されたヘッダーは薄い赤色になり、「休日」ラベルが表示されます。

### 2. 垂直方向の休日バー (オーバーレイ)
- **貫通表示**: 休日に設定された日は、全ての担当者の行を跨いで、上から下まで垂直に伸びる薄い赤色のバーが表示されます。
- **レイヤー構造**: 
    - 休日バーは工程（案件バー）よりも背面に表示されるように調整しました。
    - **休日であっても工程の設定が可能**であり、その場合は休日バーの上に案件バーが重なって表示されます。
- **実装技術**: CSS Gridの `grid-row` 指定を活用し、動的に全行をカバーする要素を生成しています。

### 3. データの永続化
- **自動保存**: 休日設定はブラウザの `localStorage` に保存され、リロードしても維持されます。
- **JSON入出力対応**: 「データを出力」ボタンで書き出したJSONファイルに休日情報が含まれ、別の環境でも復元可能です。
