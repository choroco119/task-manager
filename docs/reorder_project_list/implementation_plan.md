# 製番リストの並び替え機能の実装計画

製番（案件）リストの表示順をユーザーが自由に変更できるように、並び替え機能を追加します。現在のメンバー管理と同様、各行に「▲」「▼」ボタンを配置し、クリックで順序を入れ替えます。

## ユーザーレビューが必要な事項

> [!IMPORTANT]
> **フィルタ適用中の制限**
> 検索フィルタ（指令書No.や製品名での絞り込み）が適用されている間は、全体の中での正しい位置関係が把握しづらいため、並び替えボタンを非表示（または無効化）にします。並び替えをしたい場合はフィルタをクリアしてから行ってください。

## 検証プラン

### 手動確認
- 製番リストを開き、ボタンをクリックして行が上下に入れ替わることを確認。
- 順番を入れ替えた後、画面を更新してもその順番が維持されていることを確認。
- フィルタ入力時、並び替えボタンが適切に消えることを確認。
- 入れ替えた順番が、メイン画面の「工程割り当て」のプルダウン内容にも反映されていることを確認。
