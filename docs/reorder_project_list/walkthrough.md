# 製番リスト並び替え機能の実装完了

製番（案件）リストの表示順を自由に入れ替えられる機能を実装しました。これにより、優先度の高い案件をリストの上位に移動させるなどの操作が可能になります。

## 変更内容

### app.js
- `movePj(idx, dir)` メソッドを新規実装しました。
- `renderProjectBody()` を更新し、テーブルに「順序」列を追加しました。
- 並び替えボタン（▲▼）を各行に配置しました。
- フィルタ適用時は、全体の並び順を保護するためにボタンを非表示（ハイフン表示）にする制御を追加しました。
- 並び替え実行時に `No.`（表示上の番号）を自動的に振り直すようにしました。

### style.css
- 「順序」列 (`col-order`) の幅と文字配置のスタイルを定義しました。

## 検証結果

### 動作確認内容
1. **並び替え**: ▲▼ボタンにより、行が正しく入れ替わることを確認しました。
2. **データの永続性**: 順序を入れ替えた後、ブラウザをリロードしても順番が維持されることを確認しました。
3. **フィルタ連携**: 検索フィルタに文字を入力すると並び替えボタンが消え、フィルタをクリアすると再表示されることを確認しました。
4. **他画面への反映**: メイン画面の「工程割り当て」プルダウンの選択肢が、入れ替えた後の新しい順序で表示されることを確認しました。

> [!NOTE]
> 並び替えは、検索フィルタが空の状態（全件表示）の時のみ行えます。フィルタ適用中はボタンが非表示になりますので、順序を変えたい場合は一度検索内容をクリアしてください。
