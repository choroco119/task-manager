# 製番リストの並び替え機能（No.編集による割り込み）の実装計画

製番リストの「No.」を直接書き換えることで、その番号の位置に案件を割り込ませ、リストの順序を一括して管理できるようにします。また、この変更がメイン画面の工程表（バー内の表示）にも即座に反映されるようにします。

## ユーザーレビュー要求事項

> [!IMPORTANT]
> **並び替えのルール**
> 番号の書き換え時、既存のプロジェクトは後方にシフト（割り込み）します。
> 例: No.10 の案件を「2」に書き換えた場合、元の No.2〜9 はそれぞれ No.3〜10 に繰り下がります。
